# N-Tab-Plus

A Chrome extension for managing and organizing your browser tabs, based on the original [N-Tab](https://github.com/scoful/N-Tab) project by [scoful](https://github.com/scoful).

This is an enhanced version with additional features and bug fixes, including category management, Gist sync improvements, and UI stability fixes.

## Installation

1. Download or clone this repository
2. Open Chrome and navigate to `chrome://extensions/`
3. Enable "Developer mode"
4. Click "Load unpacked" and select the project directory
